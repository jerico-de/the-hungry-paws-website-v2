# The Hungry Paws - Pet Care Website

Welcome to **The Hungry Paws**, a pet care management website that provides grooming services, pet hotel accommodations, and pet supplies. This project is built with **Node.js, Express, MongoDB, and EJS**, with the frontend served via Express.

---

## Live Site

You can view the live website here:

[https://the-hungry-paws-website-v2.onrender.com/](https://the-hungry-paws-website-v2.onrender.com/)

---

## Features

### For All Users
- View shop information, services, and available pet products.
- Sign up and log in securely.

### For Registered Users
- Add pets with details like **breed, sex, and age**.
- Book available services for their pets.

### For Admins
- Manage bookings: **accept or reject pending requests**.
- View all user and pet information.

---

## Project Structure

- `server.js` — main Node.js/Express backend entry
- `db.js` — MongoDB connection
- `public/` — static files (HTML, CSS, JS, images)
- `views/` — EJS templates for dynamic pages
- `package.json` — dependencies and start scripts

> Note: `node_modules` and `.env` are excluded from the repository for security.

---

## How to Use

1. Open the **live site** URL in a browser.
2. You can explore the **shop info**, **services**, and **pet supplies**.
3. Register a new account or log in to start booking services.
4. Admin users can log in to view and manage pending bookings.
